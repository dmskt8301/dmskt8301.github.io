Animal Detection in Farms Using OpenCV

Built an AI Based System which monitors the field using cameras and alerts the farmer if animal intrudes into farm before it damages the crops. Proposed an AI based surveillance system to detect and monitor the presence of any animal. A camera can be placed conveniently at location(s) where any possible animal might enter from. The system uses computer vision using OpenCV to process the feed from the camera. Pre-trained model MobileNet SSD (Single Shot Detector) is used to detect the animals in the farms. The model is trained on MS COCO image dataset. A siren is fired on detecting an animal which can act as a deterrent to the animal. It can also notify the farmer so that he/she can take the concerned action as required in time


Credit Card Approval Prediction

Created a model that can forecast whether or not a credit card applicant will be accepted. The applicant's information will be used by the model to determine this. The goal is to create a model with a high degree of accuracy that the financial institution may use to make educated and effective decisions about whether to approve credit cards. Selected seven algorithms: Decision Tree, Random Forest, Multilayer Perceptrons, XGBoost, Support Vector Machines, Light GBM, Cat Boost. Out of which we will compare the accuracies to identify the best model. Form the above seven algorithms, the random forest classifier works well with the data we provided from kaggle. This will identify the fraud and legit data.
